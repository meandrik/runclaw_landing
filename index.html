// faq

Common questions.

How is my data kept safe?
Your OpenClaw agent runs with scoped permissions — it only accesses what you explicitly authorize. All connections use OAuth with minimal scopes. We don't store your emails, messages, or documents. Audit logs track every action your agent takes, and you can revoke access at any time.
Do I need to be technical to use this?
No. We handle all the technical setup — deployment, configuration, integrations, and monitoring. You just tell us what your workflow looks like and we build your agent around it. If you want to get hands-on, you can, but it's not required.
How long until my agent is live?
Most agents are deployed within 48 hours of the kickoff call. We run them in supervised mode for the first few days to tune behavior, then switch to full autonomous operation. You'll have a working agent within a week.
What tools and integrations are supported?
OpenClaw agents integrate with Gmail, Google Calendar, Slack, Notion, Linear, GitHub, and more. If you use it daily, we can likely connect it. Custom integrations are available on Growth and Scale plans.
